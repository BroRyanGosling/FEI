
## 1. Product Overview
- 一个用于记录和展示观影感受的单页应用，支持上传电影相关照片并撰写影评
- 目标用户是电影爱好者，提供沉浸式的影院风格体验，满足记录和分享观影体验的需求

## 2. Core Features

### 2.2 Feature Module
1. **编辑区**: 图片上传组件、文本输入区域
2. **预览/播放区**: 展示影评预览效果

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 主页 | 编辑区 | 支持拖拽或点击上传电影票、剧照等相关照片；提供多行文本输入框用于填写电影名称和观影感受 |
| 主页 | 预览/播放区 | 实时展示编辑区内容的最终效果，采用卡片式布局 |

## 3. Core Process
用户进入应用 → 上传观影相关照片 → 输入电影名称和影评内容 → 在预览区实时查看效果

## 4. User Interface Design
### 4.1 Design Style
- **颜色方案**: 深色模式为主（黑色/深灰背景），金色/琥珀色作为点缀色，营造电影院沉浸感
- **按钮样式**: 圆角按钮，带有微妙的悬停效果和轻微阴影
- **字体**: 标题使用粗壮醒目的字体，正文使用清晰易读的无衬线字体
- **布局风格**: 左右分栏布局，左侧编辑，右侧预览，充足留白
- **动画**: 平滑过渡和淡入效果，提升体验感

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 主页 | 编辑区 | 深色卡片、拖放上传区域、文本输入框、清晰的标签 |
| 主页 | 预览/播放区 | 电影卡片展示、标题突出显示、内容优雅排版、照片展示 |

### 4.3 Responsiveness
- 桌面端：左右分栏布局
- 移动端：上下堆叠布局，优先显示预览区
- 触摸优化：大点击区域，适合移动设备操作
